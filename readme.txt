=== Trks.it for WordPress ===
Requires at least: 3.5.0
Tested up to: 3.7.1

== Description ==

Ever wonder how many people click links that lead to 3rd party sites from your social media platforms? Trks.it is a WordPress plugin for tracking social media engagement.
